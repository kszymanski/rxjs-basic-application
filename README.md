# Very (not)advanced GitHub users search - RxJS

## Description
This project contains 2 almost the same applications that are quering GitHub API searching for users. One in Angular second in with React. Both should show you how you can work with streams in RxJS manipulating them using operators.

Remember this is example so all code is written in one component for purpose so you will see how it flows in stream to the view.

## Requrements

- [NodeJS](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/lang/en/)

## Instructions

```sh
cd streams-in-angular or streams-in-react
yarn
yarn start
```
